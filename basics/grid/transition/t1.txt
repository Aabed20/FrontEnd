Transitions :
   These effects are used to change an element from one state to another state. usually in hover effect.
    Transition contains :
       Transition-property: property-name or all 
       Transition-duration: time in seconds
       Transition-timing-function: ease/ease-in/ease-out/ease-in-out/linear

    we can achieve multiple effects using it with properties like 
     transform, filter, opacity etc..,
        1 rotate effect
         #box{
            Transition:all 2s ease;
         } 
         #box:hover{
            transform:rotate(360deg)
         } 
        2 sliding effect #box{
           Transition:all 1s ease;
          } 
          #box:hover{
            transform:translateX(200px) or transform:translateY(200px)
          }
        3 skew/elongation effect:
          #box{
            Transition:all 1s ease;
          } 
          #box:hover{
            transform:skew(20deg);
          } 
        4 zoom-in/zoom-out effect:
          #box{
            Transition:all 1s ease;
          } 
          #box:hover{
            transform:scale(1.1);
          }
